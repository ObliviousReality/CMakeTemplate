# CMakeTemplate

A basic project structure, with CMake files setup.
Compiles a basic Hello, World! program using one application and one component.

## To use

- In `./CMakeLists.txt`, replace `CMakeTemplate` with your solution name.
- Rename `sample_application` files/folders with your application name.
- Rename `sample_component` files/folders with your first component name.
- Find all lines listed with "TODO" and follow the instructions.
